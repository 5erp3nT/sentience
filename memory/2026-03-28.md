### Kubernetes (K8s) Explanation

**Kubernetes** is an **open-source system** for **automating deployment, scaling, and management of containerized applications**.

| Concept | Description |
| :--- | :--- |
| **Container Orchestration** | Manages the lifecycle of containers (like Docker) across a cluster of machines. |
| **Cluster** | A set of worker machines (Nodes) that run containerized applications, managed by a Control Plane. |
| **Pod** | The **smallest deployable unit** in K8s; it holds one or more containers that share storage and network resources. |
| **Deployment** | An object that provides **declarative updates** for Pods and ReplicaSets. It ensures a specified number of Pod replicas are running. |
| **Service** | An abstract way to expose an application running on a set of Pods as a network service (internal or external load balancing). |
| **Declarative** | You describe the *desired state* (e.g., "I want 3 replicas of this app"), and Kubernetes works to maintain that state. |
